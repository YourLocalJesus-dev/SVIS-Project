Heyo!, this is the readme for the svis project made by aravali international school,
the website contains,
-> a loading animation (first time load)
-> 4 tabs including an easter egg
-> Here are the 4 tabs explained:
  -> Tab 1 [CMD] or command center, it is a central dashboard type interface with the session (local w/o login) and a quote then below we have on the left side pending missions (or study related tasks) and on the right side there's a clock which exactly goes 
     parallel to your system clock and shows the time zone too,
     below that we have shortcuts,
     then below there we have a command prompt type interface you can type "help" and discover things and switch pages using it too, you can focus in the command prompt with "/" and find commands to use with "help" and to open the secret arcade you can type
     the command "arcade" or "matrix" on phone to unlock (use arcade on pc to get a specific order of keys to press to unlock it)
  -> Tab 2 [MSN] or Mission Log, this tab has a task management system where you can add tasks on the basis of the subject and what you have to do in it (you can also add misc) and you can also sort them by completed, non completed, and subject or topic wise,
     & also priority wise.
  -> Tab 3 [FCS] or Focus Protocol, It has a timer which you can set according to your needs on the basis of three categories
  -> Tab 4 [ARC] or Data Archive, shows your activity and trophies
  -> THE DESKTOP ICON, it helps to remove the background if the website shows some sort of performance issues. [crt mode toggle]
[EASTER EGG]
[ARCADE] it has a precision clicking game based on score, if you click continously gaining or losing score the timer pauses but if left idle the timer continues, you have to try and get a good score
and thats it for our website if you want the link
